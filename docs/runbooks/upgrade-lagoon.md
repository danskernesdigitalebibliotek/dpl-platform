# Upgrade Lagoon

## When to use

When Lagoon needs to be upgraded

## Procedure

1. Go to the [Lagoon documentation](https://docs.lagoon.sh) and find the
releases section in the right hand side of the page. Find the version you want
to upgrade to and go to it.
2. Read through the `Release notes` and keep an eye out for changes that are
out of the ordinary as well as special instructions for this release.
3. Open both the `lagoon-core-chart` and `lagoon-remote-chart` links. These
may contain extra information. They do not always have added information.
4. Now select which chart to upgrade first. This may be informed by instructions
or information from the previous 2 items.
5. Open the runbook relating to upgrading the chart chosen ie. Lagoon Remote or
Lagoon Core and run them through.
6. After upgrading both Charts we should now verify the upgrade. Open the
`Post Lagoon upgrade checklist` and follow the steps described there.

The upgrade of Lagoon should now be comlete.
